SYSADMIN

A tech-themed Warioware style game

How to Play

You have 5 live and each round you must face a new minigame, beat all 3 minigames to win

Minigames

1. Platformer: Collect all 3 CPUs before time runs out
2. Clicker: Click all 4 CPUs as fast as you can!
3. Atari: Keep the ball up for the entire duration of the timer


Controls

Arrow keys for movement, Space to jump
Mouse is primary tool for all other minigames
